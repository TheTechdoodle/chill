# Chill
A website made to relax.
Features animations, automatic sound effects (that can optionally be adjusted), and music

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
